Flex-Wrap in Modern Layouts

This page demonstrates the practical use of the CSS flex-wrap property in different scenarios. Each example shows how items behave when wrapped, not wrapped, wrapped in reverse, or combined with other Flexbox properties like justify-content or flex-direction. It serves as a visual guide and code reference for understanding how flex containers react when content exceeds the size of the container.

Basic flex-wrap Example

Introduces flex-wrap, which controls how flex items behave when they overflow the container. Items wrap onto multiple lines rather than overflowing.

HTML

Copy

 

<div class="container bg-prussian-blue"> 
    <div class="display-flex flex-col flex-wrap gap-1 m-0 p-0 w-90p h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>

flex: nowrap

This prevents wrapping even if items overflow the container. All items are forced to stay in a single row or column, possibly overflowing.

HTML

Copy

 

<div class="container bg-prussian-blue"> 
    <div class="display-flex flex-col flex-nowrap m-0 p-0 w-90p h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>

flex-wrap: reverse

Items wrap, but in the reverse order. This can be used for mirrored or inverted layouts where new items appear at the start instead of the end.

HTML

Copy

 

<div class="container bg-prussian-blue"> 
    <div class="display-flex flex-col flex-wrap-reverse m-0 p-0 w-90p h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>

justify-content: space-evenly + flex-wrap-reverse

Combines reverse wrapping with even spacing between elements. This layout ensures equal gaps between all items and wraps in reverse.

HTML

Copy

 

<div class="container bg-prussian-blue"> 
    <div class="display-flex flex-col justify-space-evenly flex-wrap-reverse m-0 p-0 w-90p h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>

flex-direction: row + flex-wrap-reverse

Reverses the wrap direction while maintaining row-wise alignment. Ideal for horizontal layouts where items should wrap from right to left or bottom to top.

HTML

Copy

 

<div class="container bg-prussian-blue"> 
    <div class="display-flex flex-row flex-wrap-reverse m-0 p-0 w-90p h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>
