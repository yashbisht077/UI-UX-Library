Flexbox Layout with Alignment Examples

Flexbox is a one-dimensional layout model that allows you to arrange items in a row or column using various alignment properties.

Below are different flexbox combinations using utility classes.

Example 1: Row Direction + Justify Center + Align Start

About It

Items are laid out in a row, horizontally centered, and aligned at the start vertically (top of the container).

HTML

Copy

 

<div class="container bg-prussian-blue w-90p" style="border: 5px solid #003049;">
    <div class="display-flex flex-row justify-center align-start m-0 p-0 h-25">
        <div class="bg-b5fccd w-8 h-8"></div>
        <div class="bg-7AC6D2 w-8 h-8"></div>
        <div class="bg-3D90D7 w-8 h-8"></div>
        <div class="bg-3674B5 w-8 h-8"></div>
        <div class="bg-3A59D1 w-8 h-8"></div>
    </div>
</div>
